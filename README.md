# 🐱 Кот-оракул

Интерактивный веб-кот, который отвечает на вопросы с помощью цитат из книги и случайной “оракульной логики”.

## 🌐 Живая версия
👉 https://cat-oracle-production.up.railway.app

## ⚙️ Функции
- Ответы кота-оракула  
- Использование текста из книги  
- Случайное “предсказание”  
- Изменение настроения кота  
- Редкий режим ИИ (Ollama)

## 💬 Идея
Проект сделан как экспериментальный “живой кот-ИИ”, который имитирует предсказания и разговор.

## 🧠 Технологии
- Python (Flask)
- HTML / JS
- Railway (деплой)
- GitHub (хранение кода)
